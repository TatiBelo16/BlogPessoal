10 Dicas para ter um site de sucesso

Abaixo, listamos 10 dicas para você construir um site usando os melhores parâmetros para uma empreitada bem sucedida:

1- Construa sites mais leves e rápidos – manter conteúdo estático em cache, otimizar a página para os browsers, manter o código limpo, reduzir excessos de mídias. São prioridades durante o processo de desenvolvimento;

2- Separe informação com menus e submenus – separar as informações de uma página de forma hierárquica, de algum termo mais geral para outro mais específico. Nada de colocar um guarda-chuva imenso de itens que com certeza não caberão na tela de um smartphone;

4- Trate fotos antes de publicá-las – imagens com alta resolução prejudicam o carregamento da página. Trabalhe para que as fotos que atendam às necessidades do projeto. Se puder abrir mão de imagens pesadas, melhor! Para o portfólio de um fotógrafo ou de uma agência de publicidade ter boas fotos não tem jeito. Nestes casos, uma opção é o uso de algumas ferramentas de compressão;

5- Adote um visual harmônico – cores fortes estão permitidas. Mas é preciso conhecer as combinações e atender aos objetivos do projeto. Se não tiver certeza, fique na linha do clean. O risco de errar é menor;
6 – A navegação deve seguir uma ordem lógica – pensar nas trajetórias do usuário pela página faz parte de estudos para desenvolvimento de interfaces;

7- Cuide dos títulos das páginas – elas ajudam os buscadores a encontrarem seu site e favorecerem o SEO. Leve em consideração que o Google dá preferência para conteúdos relevantes, então cuide do seu texto, prefira os que tem mais de 500 caracteres e repita algumas vezes as palavras-chaves mais importantes para tentar melhorar o posicionamento orgânico;

8- Use apenas um tipo de fonte em todo o website – se quiser mais de uma, use apenas para destaques, citações e títulos. Assim o visual fica mais limpo e padronizado;

9- Códigos devem ser escritos manualmente ou por programas específicos, como Dreamweaver – um processador de textos qualquer poderá criar códigos desnecessários;

10- Espaço em branco ajuda a deixar o conteúdo legível – mas não exagere, para não comprometer o design.

Autor: Gustavo Guanabara
Publicação: 19/09/2017
